# deep learning from scratch nlp

## setup environment

Ubuntu

```shell
sudo apt install libeigen3-dev
```

Mac

```shell
brew install eigen
```

## compile

Ubuntu

```shell
g++ -I /usr/include/eigen3 main.py
```

Mac

```shell
g++ -I /usr/local/include/eigen3 main.cpp
```

## run

```shell
./a.out
```
